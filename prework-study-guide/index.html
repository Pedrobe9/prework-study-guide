HTML

    • The head element contains information about the webpage.
    • Headings are size h1 (biggest) up to h6 (smallest).
    • Title element in head appears in tab.
    • style and base element defines the base URL for a webpage.
    • link, meta and script element embeds client-side scripts in the HTML document.
    • The body element represents the visible content shown to the user.
    • Header element appears at the top of the page and usually contains a navigation menu and/or introductory content. In our case, it holds an h1 heading, up to h6.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch; -A: add all changes.
  • git commit -m "add starter code": commit message
  • git pull origin main: receive a branch's modifications into the local environment
